# Hooks Folder

This folder contains custom React hooks used throughout the application.

## Structure

The hooks folder is structured as follows:

Each file exports a custom React hook. For example, `useCustomHook1.js` exports a hook named `useCustomHook1`.

## Usage

To use a custom hook, import it in your component like so:

```javascript
import { useCustomHook1 } from './hooks/useCustomHook1';
```
